# Currency Converter

A React-based currency converter that fetches real-time exchange rates and allows users to convert between different currencies.

## Tech Stack
- React
- JavaScript
- CSS

## Features
- Convert between multiple currencies
- Real-time exchange rates
- Simple and intuitive UI

## Note
This project was built to practice React hooks, API integration, and state management.
